# ps-kModule ![Language](https://img.shields.io/badge/language-c-blue.svg?style=flat-square)

以内核Module的方式遍历当前系统所有进程描述符，并以树状结构显示

## 使用

* 编译、insmod	： make
* 查看输出信息	： dmesg
* clean、rsmod	： make clean

## 若以系统调用的方式显示需要移植到Linux内核源码对应位置
